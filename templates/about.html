About the Project

Smart Agricultural Yield Prediction

This application uses machine learning to predict agricultural crop yields based on historical data and various parameters such as location, crop type, and farming inputs.

The system employs an XGBoost regression model trained on extensive agricultural data from multiple regions of India. Our model analyzes patterns in historical crop yields to provide accurate predictions for new input parameters.

How to Use

To use this application:

  1. Navigate to the home page and locate the prediction form
  2. Select the state, district, crop type, and season
  3. Enter the crop year, area (in hectares), and expected production (in tonnes)
  4. Click "Predict Yield" to generate a prediction
  5. View the results, which include the predicted yield and visualizations
